Institutional What If

Here we explore the impacts of what if questions on an institution. That is, how many majors/plans do changes in the prerequisite structure affect?

Add a Course

To add a course, please specify the new name and the credit hours.

Strictly speaking adding prereqs and dependencies is not required, but courses that have no dependencies will not affect any plans other than the ones that they are manually added to.

For dependency and prerequisite course names, please make sure the name matches perfectly. If you're looking for "MATH 20C" please type that out, not "MATH20C" or "MATH 20C"

Select the plans this class will be in

Add a Prerequisite

Here you can add a prerequisite. Please fill in the course you want to add a prerequisite to and the course you want to add as the prerequisite

For course names, please make sure the name matches perfectly. If you're looking for "MATH 20C" please type that out, not "MATH20C" or "MATH 20C"

Remove a course

Here you can remove a course. Please simply specify the name of the course to remove

For course names, please make sure the name matches perfectly. If you're looking for "MATH 20C" please type that out, not "MATH20C" or "MATH 20C"

Remove a prerequisite

Here you can remove a prerequisite. Please specify the course that will lose a prerequisite and the prerequisite it will lose.
